# 小米官网静态复刻项目

*一个响应式的小米官网首页静态复刻*

##  项目简介

使用纯前端技术实现的静态小米官网首页复刻项目，较完美还原了小米官网的视觉风格和主要功能模块。

##  核心功能

###  页面布局
- 1:1 还原小米官网的视觉设计
- 完整的页面结构：头部、导航、轮播、产品区、页脚
- 像素级还原小米的UI细节


## 🛠️ 技术实现

| 技术        | 用途                          | 备注                     |
|-------------|-------------------------------|--------------------------|
| HTML5       | 页面语义化结构                | 使用W3C标准             |
| CSS3        | 样式与动画                    | Flex/Grid布局           |
| JavaScript  | 交互逻辑实现                  | 原生JS实现              |
